Truck Purchase Intention Collection Platform

Author: XIE LIDONG

Project Overview
Streamlit-based bilingual (Arabic/English) web app for truck purchase inquiries, with Feishu notifications and PDF brochure downloads.
Technology Stack
Python, Streamlit, Feishu Webhook, Pytz, PDF Handling, Form Validation
Key Features
Bilingual UI with custom styling
Truck model selection + mandatory contact info validation
Auto Feishu notifications (Saudi timezone timestamps)
PDF brochure download for selected models
Error handling for submissions/files
